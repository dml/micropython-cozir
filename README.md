# MicroPython CoZiR

COZIR CO2 Sensor Library

# Installation

Install on the Unix port

```bash
micropython -m upip install -p . micropython-cozir
```

Install on the device port via REPL

```python
import upip
upip.install("micropython-cozir")
```
